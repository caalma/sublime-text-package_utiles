# Utilidades personalizadas para Sublime-Text 


## Modo de uso

Copiar la carpeta "./utiles-sublime-package/":

+ en Linux, a: "~/.config/sublime-text-3/Packages/"
+ en Windows, a:  "%AppData%\Roaming\Sublime Text 3\Packages"
